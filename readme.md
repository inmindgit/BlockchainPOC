Dapp Test sobre Smart Contracts en Etherium

- Correr Proyecto

- Instalar npm install -g ethereumjs-testrpc
- En consola correr testrpc
- Instalar paquetes de proyecto
    - npm install
- Ejecutar comando de Truffle
    - truffle compile
    - truffle migrate
- Correr proyecto
    - npm run start
- Correr test
    - truffle test
