Dapp Test sobre Smart Contracts en Etherium

- Correr Proyecto

1 - Instalar npm install -g ethereumjs-testrpc
2 - En consola correr testrpc
3 - Instalar paquetes de proyecto
    - npm install
4 - Ejecutar comando de Truffle
    - truffle compile
    - truffle migrate
5 - Correr proyecto
    - npm run start
6 - Correr test
    - truffle test
